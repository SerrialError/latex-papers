## Important interpretation rule

CEA/RocketCEA vacuum Isp and CEA's exit-pressure-matched Isp are not automatically sea-level Isp for an arbitrary nozzle. For sea-level sizing use the `isp_amb_s` column, which applies the ambient pressure correction and reports separation/overexpansion mode.
